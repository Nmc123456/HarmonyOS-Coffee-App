# HarmonyOS 仿瑞幸咖啡App  
![咖啡App界面](https://github.com/Nmc123456/HarmonyOS-Coffee-App/blob/main/harmonyos-luckin-coffee-app1.png?raw=true)  

## 项目简介  
- **功能**：用户登录、商品展示、购物车、订单管理  
- **技术栈**：HarmonyOS 3.0、Java、分布式数据管理  
- **亮点**：实现跨设备购物车同步，优化界面响应速度40%
